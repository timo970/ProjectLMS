ПОЯСНИТЕЛЬНАЯ ЗАПИСКА К ПРОЕКТУ

Тема проекта: Веб-сайт игрового проекта «The UnderWorld» с системой авторизации
Разработчики: Тимофей и Константин

1. ОПИСАНИЕ ВЕБ-САЙТА
Веб-сайт разработан как официальная информационная и дистрибьюторская платформа для 2D-аркады «The UnderWorld». На главной странице сайта представлены:
- Общая информация об игровом процессе и аудиодизайне.
- Ссылки и кнопки для скачивания версий игры под операционные системы Windows 10/11 и Linux.
- Блок отзывов активных игроков.
- Краткое руководство (гайд) по выживанию для новичков.

2. АРХИТЕКТУРА И ЛОГИКА ИНТЕРФЕЙСА (UX/UI)
Сайт оснащен динамической системой взаимодействия с пользователем:
- В верхнем левом углу главной страницы размещена интерактивная иконка личного кабинета.
- При клике на иконку открывается всплывающее меню / форма авторизации («Войти»).
- Форма содержит стандартные поля ввода данных: Email (в качестве логина) и Пароль.
- Внутри формы авторизации предусмотрена гиперссылка «Зарегистрироваться» для новых пользователей.
- При клике на ссылку «Зарегистрироваться» происходит перенаправление (редирект) на отдельную веб-страницу с формой регистрации нового аккаунта.

3. ТЕХНИЧЕСКАЯ ЧАСТЬ И БАЗА ДАННЫХ
Для обеспечения работы личного кабинета реализована серверная логика и интеграция с базой данных:
- В качестве системы управления базами данных (СУБД) используется библиотека SQLite3.
- База данных хранит учетные записи пользователей в специальной таблице.
- При регистрации данные (Email и Пароль) отправляются с веб-формы и записываются в БД.
- При авторизации система выполняет SQL-запрос к SQLite3 для проверки совпадения введенной пары Email/Пароль с данными в базе. Если данные верны, пользователь успешно входит в систему.

4. ЗАКЛЮЧЕНИЕ
Разработанный веб-сайт успешно выполняет функции презентации игрового продукта, обеспечивает скачивание дистрибутивов и предоставляет безопасный интерфейс для управления учетными записями пользователей на базе локальной СУБД SQLite3.
